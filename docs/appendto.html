Private method mix

Mixing objects to a new combined object, does not clone

Parameters:

  • defaults must be an Object.
    (default options)

  • overrides must be an Object.
    (Overrides)

Returns an Object
(a mix of the default and overrides)

Example:

mix({ fun : true },{});
=> { fun : true }
mix({ fun : true },{ fun : false });
=> { fun : false }
mix({ fun : true },{ dead : false });
=> { fun : true, dead : false }
function mix(defaults, overrides) { var result = {}; var k; for (k in defaults) { if (defaults.hasOwnProperty(k)) { result[k] = defaults[k]; } } for (k in source) { if (source.hasOwnProperty(k)) { result[k] = overrides[k]; } } return result; }
